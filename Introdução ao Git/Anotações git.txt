git init (cria o repositório)

git add "nome do arquivo" (Add o arquivo ao stage)
git add . ou * (para adicionar a stage tudo)

git commit -m "mensagem do commit" (para commitar)

git push origin main (para empurrar o arquivo para o github)

git pull origin main (para consertar erro como  mergin)

git clone "url" (para clonar repositório)

git remote -v (lista os repositórios remotos)
